Title: Интерпретатор языка C++ &#8211; Cling
Slug: cling-c-interpreter
Categories: techie,tutorials
<p>Не так давно наткнулся на один интересный проект под названием <strong>Cling</strong>, интерпретатор языка C++, который может работать в интерактивном режиме из консоли в том числе. Ознакомиться с проектом можно по ссылке: <a href="https://github.com/root-project/cling" target="_blank" rel="noopener noreferrer">https://github.com/root-project/cling</a><br />
Установка для Ubuntu простейшая &#8211; скачать архив для нужно версии, распаковать, зайти в папку bin и запустить cling в терминале.<br />
