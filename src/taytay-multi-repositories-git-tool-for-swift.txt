<p>Представляю вашему вниманию <a href="https://gitlab.com/demensdeum/taytay" target="_blank" rel="noopener">Taytay</a> &#8211; утилиту для контроля статуса репозиторев git для языка Swift. На текущий момент Свифт можно установить на всех мейнстримовых десктопных операционных системах. Для Ubuntu я рекомендую использовать <a href="https://github.com/kylef/swiftenv" target="_blank" rel="noopener">Swiftenv</a>. Завязан <a href="https://gitlab.com/demensdeum/taytay" target="_blank" rel="noopener">Тайтай</a> на утилиту <a href="https://github.com/git-cola/git-cola" target="_blank" rel="noopener">git-cola</a>, но можешь отредактировать исходник и поменять на любую другую программу.</p>
