<p>Первый прототип игры Space Jaguar Action RPG для Webassembly:</p>
<p><a href="https://demensdeum.com/demos/SpaceJaguarActionRPG004/" target="_blank" rel="noopener"><img loading="lazy" decoding="async" class="alignnone wp-image-2897" src="https://demensdeum.com/blog/wp-content/uploads/2021/01/spaceJaguar1.jpg" alt="" width="611" height="343" srcset="https://demensdeum.com/blog/wp-content/uploads/2021/01/spaceJaguar1.jpg 1222w, https://demensdeum.com/blog/wp-content/uploads/2021/01/spaceJaguar1-300x168.jpg 300w, https://demensdeum.com/blog/wp-content/uploads/2021/01/spaceJaguar1-1024x575.jpg 1024w, https://demensdeum.com/blog/wp-content/uploads/2021/01/spaceJaguar1-768x431.jpg 768w" sizes="auto, (max-width: 611px) 100vw, 611px" /></a></p>
<p><em>Загружаться будет долго (53мб) без индикации загрузки.</em></p>
<p>Space Jaguar Action RPG &#8211; симулятор жизни космического пирата. На данный момент доступны простейшие игровые механики:</p>
<ul>
<li>летать в космосе</li>
<li>умирать</li>
<li>кушать</li>
<li>спать</li>
<li>нанимать команду</li>
<li>смотреть на неугомонный, быстролетящий поток времени</li>
</ul>
<p>Из-за плохой оптимизации отрисовки 3D сцен в веб версии, недоступна возможность гулять в трехмерном окружении. Оптимизация будет добавлена в следующих версиях.</p>
<p>Исходный код движка, игры, скриптов доступен по лицензии MIT. Все предложения по улучшению принимаются крайне позитивно:</p>
<p><a href="https://gitlab.com/demensdeum/space-jaguar-action-rpg" target="_blank" rel="noopener">https://gitlab.com/demensdeum/space-jaguar-action-rpg</a></p>
